# 🚀 AI Smart Surveillance System with Voice Alert

> A real-time intelligent surveillance system that detects motion and triggers voice alerts with automatic image capture.

---

## 📌 Overview

This project is a **real-time AI-based surveillance system** built using Python and OpenCV.  
It detects motion through frame comparison, generates **instant voice alerts**, and captures images whenever unusual activity is detected.

This project demonstrates practical implementation of **computer vision + automation**, making it useful for real-world security applications.

---

## 🎯 Features

✅ Real-time motion detection  
✅ Instant voice alert using Text-to-Speech  
✅ Automatic image capture on detection  
✅ Lightweight & fast (no heavy ML models required)  
✅ Works with webcam (live monitoring)  

---

## 🛠️ Tech Stack

- Python  
- OpenCV  
- pyttsx3 (Text-to-Speech)
